# Salesforce 製品別の標準オブジェクト

Salesforce の標準オブジェクトは、製品ごとに完全に分かれているわけではありません。`Account`、`Contact`、`User`、`Task`、`Event`、`Product2` などは複数の製品領域で共通して使われます。

このディレクトリでは、製品ごとに 1 ページで、よく確認する標準オブジェクトをラベル、API 名、役割で整理します。組織のライセンス、機能有効化、権限、導入パッケージによって利用できるオブジェクトは変わります。

## 読み方

同じオブジェクトが複数ページに出てくる場合は、その製品領域でも主要な役割を持つという意味です。

製品固有の業界オブジェクトや管理パッケージ由来のオブジェクトは、組織や導入方式によって API 名が異なる場合があります。標準オブジェクトの最終確認は公式 Object Reference と対象組織の Object Manager で行います。
