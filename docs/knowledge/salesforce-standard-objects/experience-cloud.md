# Experience Cloud の標準オブジェクト

Experience Cloud はサイト、ポータル、コミュニティの入口として使われ、業務データは Sales / Service / Commerce のオブジェクトを共有することが多いです。

製品ページ: [Experience Cloud](https://www.salesforce.com/experience-cloud/)

| ラベル         | API 名                    | 役割                             |
| -------------- | ------------------------- | -------------------------------- |
| 取引先         | `Account`                 | 顧客企業、パートナー、会員組織   |
| 取引先責任者   | `Contact`                 | 外部ユーザーに紐づく人物         |
| ユーザー       | `User`                    | 外部ユーザー、社内ユーザー       |
| ケース         | `Case`                    | ポータルから登録される問い合わせ |
| ナレッジ       | `KnowledgeArticleVersion` | 公開 FAQ、ヘルプ記事             |
| ファイル       | `ContentDocument`         | 公開、共有されるファイル         |
| ファイルリンク | `ContentDocumentLink`     | ファイルとレコードの関連         |
| フィード項目   | `FeedItem`                | サイト上の投稿、フィード         |
| 注文           | `Order`                   | ポータル上で参照する注文         |
| 納入商品       | `Asset`                   | 顧客が参照する保有商品           |
