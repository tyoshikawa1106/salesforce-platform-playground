# Marketing の標準オブジェクト

Salesforce Platform 上で見る Marketing 関連の標準オブジェクトは、CRM 側のキャンペーン管理が中心です。Marketing Cloud Engagement や Account Engagement は、別体系のデータモデルやコネクタ経由のデータも扱います。

製品ページ: [Marketing Cloud](https://www.salesforce.com/marketing/)

| ラベル               | API 名           | 役割                                     |
| -------------------- | ---------------- | ---------------------------------------- |
| キャンペーン         | `Campaign`       | マーケティング施策、リード獲得活動       |
| キャンペーンメンバー | `CampaignMember` | キャンペーンとリード、取引先責任者の関係 |
| リード               | `Lead`           | 見込み客                                 |
| 取引先責任者         | `Contact`        | 顧客担当者、配信対象                     |
| 取引先               | `Account`        | 顧客企業、セグメント                     |
| メールテンプレート   | `EmailTemplate`  | メール文面テンプレート                   |
| ToDo                 | `Task`           | フォローアップ                           |
