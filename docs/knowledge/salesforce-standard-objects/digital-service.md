# Digital Service の標準オブジェクト

Digital Service は、メッセージング、チャット、電話、ケース連携を中心に扱います。

製品ページ: [Service Cloud](https://www.salesforce.com/service/)

| ラベル                   | API 名               | 役割                                   |
| ------------------------ | -------------------- | -------------------------------------- |
| メッセージングセッション | `MessagingSession`   | デジタルチャネルの会話セッション       |
| チャット記録             | `LiveChatTranscript` | Web チャットの会話履歴                 |
| 音声通話                 | `VoiceCall`          | 電話連携、コンタクトセンター通話       |
| ケース                   | `Case`               | 会話から作成、関連付けされる問い合わせ |
| 取引先責任者             | `Contact`            | 問い合わせ元の人物                     |
| 取引先                   | `Account`            | 問い合わせ元の顧客企業                 |
| ユーザー                 | `User`               | オペレーター、担当者                   |
| メールメッセージ         | `EmailMessage`       | メールチャネルの本文と履歴             |
