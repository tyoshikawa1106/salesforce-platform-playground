# Consumer Goods Cloud の標準オブジェクト

Consumer Goods Cloud は、店舗、訪問、商品、取引先を中心に扱います。組織や導入方式によって、業界固有オブジェクトの API 名や名前空間が異なる場合があります。

製品ページ: [Consumer Goods Cloud](https://www.salesforce.com/consumer-goods/)

| ラベル       | API 名        | 役割                         |
| ------------ | ------------- | ---------------------------- |
| 取引先       | `Account`     | 小売企業、店舗運営会社       |
| 取引先責任者 | `Contact`     | 店舗担当者、取引先担当者     |
| 店舗         | `RetailStore` | 訪問、棚割、活動対象店舗     |
| 訪問         | `Visit`       | 店舗訪問、営業活動           |
| 商品         | `Product2`    | 店頭商品、販売商品           |
| 価格表       | `Pricebook2`  | 商品価格のまとまり           |
| ToDo         | `Task`        | 店舗訪問後の作業             |
| 行動         | `Event`       | 訪問予定、商談予定           |
| ケース       | `Case`        | 店舗、商品に関する問い合わせ |
