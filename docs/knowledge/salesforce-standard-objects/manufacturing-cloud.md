# Manufacturing Cloud の標準オブジェクト

Manufacturing Cloud は、顧客、商品、販売計画、注文を中心に扱います。組織や導入方式によって、業界固有オブジェクトの API 名や名前空間が異なる場合があります。

製品ページ: [Manufacturing Cloud](https://www.salesforce.com/manufacturing/)

| ラベル       | API 名           | 役割                     |
| ------------ | ---------------- | ------------------------ |
| 取引先       | `Account`        | 顧客企業、販売先         |
| 取引先責任者 | `Contact`        | 顧客側の担当者           |
| 商品         | `Product2`       | 製造、販売対象の商品     |
| 価格表       | `Pricebook2`     | 商品価格のまとまり       |
| 商談         | `Opportunity`    | 案件、需要、売上見込み   |
| 販売契約     | `SalesAgreement` | 販売計画、契約数量、予測 |
| 注文         | `Order`          | 受注、出荷につながる注文 |
| 注文商品     | `OrderItem`      | 注文に含まれる商品明細   |
| ケース       | `Case`           | 顧客問い合わせ、品質対応 |
| 納入商品     | `Asset`          | 顧客が保有、利用する商品 |
