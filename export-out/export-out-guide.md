# export-out guide

`export-out/` is the local output directory for Salesforce CLI export files.

CSV and JSON files generated by `sf data export bulk` are not tracked by Git.
Keep this guide tracked so the export output directory name stays consistent.

## CSV

```sh
sf data export bulk \
  --query-file scripts/soql/setup-data/accounts.soql \
  --output-file export-out/accounts.csv \
  --result-format csv \
  --wait 30 \
  --target-org <alias>
```

## JSON

```sh
sf data export bulk \
  --query-file scripts/soql/setup-data/accounts.soql \
  --output-file export-out/accounts.json \
  --result-format json \
  --wait 30 \
  --target-org <alias>
```

See [test-data-import.md](../docs/development/test-data-import.md) for the
standard object setup workflow.
